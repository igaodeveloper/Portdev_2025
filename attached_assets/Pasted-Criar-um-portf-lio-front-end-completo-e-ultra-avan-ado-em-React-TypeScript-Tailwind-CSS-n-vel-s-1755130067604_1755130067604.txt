Criar um portfólio front-end completo e ultra avançado em React + TypeScript + Tailwind CSS, nível sênior, extremo e imersivo, que surpreenda qualquer recrutador. O portfólio deve ter UX/UI absurdamente inovador, com Dark Mode, parallax cósmico, partículas, glow, TV glitch, fly-through space e microinterações.

1️⃣ Paleta de cores

Preto profundo (primário): #000000

Roxo clássico: #6B4CFF

Roxo vívido (destaque): #9B30FF

Branco (textos e ícones): #FFFFFF

Cinza claro (backgrounds secundários): #E0E0E0

Azul espacial (destaque cósmico/planetas/partículas): #3AB0FF

2️⃣ Tecnologias e bibliotecas

React + TypeScript

Tailwind CSS

Framer Motion / React Spring (animações avançadas)

Three.js / react-three-fiber (planetas, nebulosas 3D, universo imersivo)

Particles.js / tsparticles (estrelas, partículas, efeitos cósmicos)

Parallax.js / Rellax.js (parallax avançado multi-camadas)

Monaco Editor ou Sandpack (IDE estilo VSCode)

xterm.js (terminal interativo)

Axios / Fetch API (YouTube, outros APIs)

Zustand / React Context API (estado global)

Deploy: Vercel / Netlify
