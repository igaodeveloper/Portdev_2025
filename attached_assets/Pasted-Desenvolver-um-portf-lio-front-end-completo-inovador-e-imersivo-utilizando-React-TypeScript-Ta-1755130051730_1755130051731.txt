Desenvolver um portfólio front-end completo, inovador e imersivo, utilizando React + TypeScript + Tailwind CSS, com UX/UI de altíssimo nível, Dark Mode, animações cósmicas e funcionalidades avançadas.
O portfólio deve ser robusto, modular e profissional, surpreendendo qualquer recrutador.

3️⃣ Tecnologias e Bibliotecas

React + TypeScript

Tailwind CSS

Framer Motion / React Spring – animações avançadas

Three.js / react-three-fiber – planetas, estrelas, nebulosas 3D

Particles.js / tsparticles – partículas interativas

Parallax.js / Rellax.js – parallax multi-camadas

Monaco Editor / Sandpack – IDE estilo VS Code

xterm.js – terminal interativo

Axios / Fetch API – YouTube e outras APIs

Zustand / React Context – estado global
